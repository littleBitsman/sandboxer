## Testing

This project includes a comprehensive test suite covering all critical functionalities. The test suite validates:

- Instance allow/disallow logic
- Instance and signal wrapping/unwrapping
- Sandbox initialization and configuration
- Environment isolation and security restrictions
- Edge cases and boundary conditions

See [tests/README.md](tests/README.md) for detailed information about running and contributing to the test suite.
